＃作りたいもの
・VMとかバイトコードを作る
・ちょっと関数型チックにしたい

＃文法
program     =   expr *
expr        =   assign
assign      =   equality ("=" assign) ?
equality    =   relational ("==" relational | "!=" relational ) *
relational  =   add ("<" add | "<=" add | ">" add | ">=" add)*
add         =   mul ("+" mul | "-" mul)*
mul         =   unary ("*" unary | "/" unary)*
unary       =   ("+" | "-")? primary
primary     =   num | ident | "(" expr ")"

＃現在の目標
変数を使えるようにする
「return」キーワード

＃達成済み
四則演算
丸括弧
単項の加減算
(現状)トークンを区切る空白の有無が任意
字句解析中にunknown tokenがあれば報告

＃とりあえず
バイトコードは置いといて、ひとまずアセンブリを吐く
